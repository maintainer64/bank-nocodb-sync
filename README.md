# Bank Sure Sync

Расширение для браузера Chrome, которое автоматически синхронизирует счета и транзакции из банковских приложений с
**Sure** или выгружает в CSV файлы.

## Поддерживаемые сервисы

- ✅ **Т-Банк**
- ✅ **Яндекс Банк**
- ✅ **Яндекс Лавка**
- ✅ **Жизнь Март**

## Функциональность

- 🔄 **Автоматическая синхронизация** счетов и транзакций в Sure
- 📁 **Выгрузка в CSV** для локального хранения
- 🔒 **Безопасное хранение** настроек и данных
- 🎯 **Простой интерфейс** настройки и управления

## Установка

### Способ 1: Автоматическая установка (рекомендуется)

1. Скачайте и запустите скрипт установки:

```bash
curl -s https://raw.githubusercontent.com/maintainer64/bank-nocodb-sync/main/install.sh | bash
```

Или вручную:

```bash
# Скачайте скрипт
wget https://github.com/maintainer64/bank-nocodb-sync/releases/latest/download/install.sh

# Дайте права на выполнение
chmod +x install.sh

# Запустите установку
./install.sh
```

### Способ 2: Ручная установка

1. **Скачайте последний релиз**:

```bash
# Используйте скрипт для скачивания
./install.sh
```

2. **Распакуйте расширение**:
    - Архив автоматически распакуется в `~/bank-nocodb-sync_latest/`

3. **Добавьте расширение в Chrome**:
    - Откройте Chrome и перейдите в `chrome://extensions/`
    - Включите **"Режим разработчика"** (переключатель в правом верхнем углу)
    - Нажмите **"Загрузить распакованное расширение"**
    - Выберите папку с распакованным расширением

## Настройка

1. Нажмите на иконку расширения в Chrome
2. Перейдите в настройки
3. В разделе **Настройки Sure** укажите базовый адрес вашего экземпляра Sure

## Использование

### Синхронизация с Sure

1. Откройте нужный банковский сервис в браузере
2. Нажмите на иконку расширения
3. Нажмите **"Синхронизировать в Sure"**
4. Счета и транзакции будут автоматически созданы или обновлены в Sure

### Ручной экспорт в CSV

1. Откройте нужный банковский сервис
2. Нажмите на иконку расширения
3. Выберите:
   - **"Операции в CSV"** — экспорт транзакций
   - **"Счета в CSV"** — экспорт счетов
   - **"Заказы в CSV"** — экспорт заказов (для маркетплейсов)

## Разработка

### Требования

- Node.js 20+
- Yarn 1.x

### Установка для разработки

```bash
# Клонируйте репозиторий
git clone https://github.com/maintainer64/bank-nocodb-sync.git
cd bank-nocodb-sync

# Установите зависимости
yarn install

# Запустите в режиме разработки
yarn dev

# Соберите для production
yarn build
```

### Структура проекта

```
src/
├── index.tsx          # Основной компонент
├── background/        # Фоновые скрипты
├── content/          # Контент-скрипты для банковских сервисов
├── utils/            # Утилиты и хелперы
└── types/            # TypeScript типы
```

## Безопасность

- 🔐 **Локальное хранение** — все настройки хранятся локально в браузере
- 🌐 **Безопасные запросы** — использование HTTPS для API вызовов

## Troubleshooting

### Расширение не работает

1. Убедитесь, что расширение включено в `chrome://extensions/`
2. Проверьте настройки Sure (базовый адрес)
3. Откройте консоль разработчика (F12) для просмотра ошибок

### Данные не синхронизируются

1. Проверьте подключение к интернету
2. Убедитесь, что сервер Sure доступен
3. Проверьте правильность адреса Sure в настройках
